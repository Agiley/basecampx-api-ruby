# Basecampx

Basecamp new API wrapper

API you can find here https://github.com/37signals/bcx-api

## Installation

Add this line to your application's Gemfile:

    gem 'basecampx'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install basecampx
